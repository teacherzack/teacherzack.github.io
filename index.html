ScratchJr家庭日

发表于 2019-10-28 | 分类于 ScratchJr介绍

ScratchJr家庭日是一个社区活动，父母、祖父母、兄弟姐妹、其他家庭成员和孩子们一起在ScratchJr里制作项目。家庭日的目的不仅是帮助成人和孩子更加熟悉ScratchJr的使用，也是鼓励成人和孩子合作制作ScratchJr项目。

谁来参加家庭日？

这个活动可以让任何想参与ScratchJr社区的人来参加！它可以在学校、博物馆、课后班、社区中心、或者任何家庭可以聚在一起的地方举办。我们鼓励5-7岁的孩子和他们的家庭成员（包括兄弟姐妹或者更大或更小的家庭成员）都来参加！

第一课 ScratchJr快速上手

发表于 2019-10-27 | 分类于 ScratchJr视频教程

教学目的

通过一个简单示例的学习，快速上手ScratchJr的基本使用。让小朋友们有成就感，对ScratchJr编程产生兴趣。

场景

  1. 沙滩
  2. 海底世界

人物

  1. Tic
  2. Toc
  3. 绿色小鱼
  4. 4.海星

剧情一

主人公Tic和Toc在海滩玩耍，Toc在高兴的不停的变大又变小，Tic走着走着跳到了大海里。

剧情二

Tic在深海中出现，高兴的说，深海世界我来啦，并时不时吐泡泡。一条绿色的鱼在Tic的头上游动，海底的海星在海水的带动下左右摇晃。

开始，移动，跳跃，有限循环，无限循环，消失，变大，变小，说话，声音，左转，右转，场景切换
