学编程究竟是在学什么

发表于 2019-10-29 | 分类于 STEAM教育

编程的过程，就是解决某个问题的过程。

程序员在编程的过程中，实际上就是使用工程学和数学方法，逐步构建了一个解决某一个或某一类问题的方法。

少儿编程的目的，就是让小朋友在类似玩游戏的过程中，逐步养成求解解决问题的思维方式。碰到一个问题，让小朋友养成这种宝贵的思维方式：这个问题看起来不简单，但是我可以解决。把一个看似复杂的问题拆解成多个小问题，首先要怎样处理，然后怎样做，我可以先这样做，不会的问题我可以通过学习解决掉。这种解决问题的思维方式和能力，无论在生活中，还是在课堂学习上，对小朋友来说都是极其宝贵的财富。

少儿编程能激发小朋友强烈的学习兴趣，在兴趣的驱使下，最大化孩子的创作欲望。从而培养孩子的创新思维和创造能力。创新和创造产生的成就感，也会让孩子更加自信。

ScratchJr是什么

发表于 2019-10-28 | 分类于 ScratchJr介绍

ScratchJr 是一个入门级的编程语言，它可以让幼儿 ( 5-7岁 ) 创建互动的故事和游戏，孩子利用图形化的程序积木让角色移动、跳跃、舞蹈、唱歌。孩子也可以利用绘图编辑器绘制自己的角色、用麦克风录制自己声音、用照相机加入自己拍摄的照片。最后用拼积木的方法组合成程序，让他们心中的角色动起来。

ScratchJr 这个软件的创作灵感来自于 Scratch。在全球有数以百万计的年轻人 ( 8 岁以上 ) 使用 Scratch，是个非常受欢迎的编程语言( scratch.mit.edu ) 。而 ScratchJr 则重新设计操作界面及编程语言，让它更适合幼儿，尤其专注将功能设计得更符合他们的认知、个人、社会、情感发展。

你可以在 iPad 或是 Android 平板上免费下载到 ScratchJr ，若需进一步了解，请参考本站其他教学内容。

编码 ( 或电脑程序 ) 是新世代的必备素养，就像写作能让你的组识能力、表达能力变好一样，编程也一样。在过去，编程很难普及，但我们认为每个人都该具备这样的能力，就像写作能力一样。

当幼儿使用 ScratchJr 时，他们不只是和电脑互动，更可以通过电脑学习如何创造和表达内心所想。从这个过程中，孩子们也学到了解决问题、创意设计、逻辑思考的能力，这都是帮助他们在未来成功的基础。同时，他们在有意义、有动力的情况下使用数学、语文，能力的发展会变得更快。ScratchJr 希望孩子们不仅仅是学习程序代码，也能从编码中学习到更多更多。
