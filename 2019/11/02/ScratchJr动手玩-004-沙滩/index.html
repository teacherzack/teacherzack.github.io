ScratchJr动手玩-004-沙滩

发表于 2019-11-02 | 分类于 视频教程-ScratchJr直接动手玩

教学目的

通过一个简单示例的学习，快速上手ScratchJr的基本使用。让小朋友们有成就感，对ScratchJr编程产生兴趣。

场景

  1. 沙滩
  2. 海底世界

角色

  1. Tic
  2. Toc
  3. 绿色小鱼
  4. 海星

剧情一

主人公Tic和Toc在海滩玩耍，Toc在高兴的不停的变大又变小，Tic走着走着跳到了大海里。

剧情二

Tic在深海中出现，高兴的说，深海世界我来啦，并时不时吐泡泡。一条绿色的鱼在Tic的头上游动，海底的海星在海水的带动下左右摇晃。

用到的功能模块

开始，移动，跳跃，有限循环，无限循环，消失，变大，变小，说话，声音，左转，右转，场景切换
